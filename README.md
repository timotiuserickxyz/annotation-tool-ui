# automlui

## Setup
### nvm / npm

```sh
% nvm install vX.Y.Z # see .nvmrc
% nvm use vX.Y.Z
```

```sh
% npm i
```

## Development
```sh
% npm run dev 
% open http://localhost:3000
```

## Linting
```sh
% npm run eslint
```

## Format
```sh
% npm run format
```

## Storybook
```sh
% npm run storybook
% open http://localhost:6006
```
